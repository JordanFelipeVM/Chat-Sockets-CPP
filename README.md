# Chat para Sala de Cómputo

Sistema de chat para comunicación entre dispositivos en una sala de cómputo.

## Características

- Comunicación mediante sockets TCP
- Almacenamiento de historial en SQLite
- Identificación única por dispositivo
- Visualización de mensajes en tiempo real
- Historial de conversaciones persistente

## Requisitos

- C++11 o superior
- SQLite3
- Make
- Sistema Unix/Linux (para sockets)
